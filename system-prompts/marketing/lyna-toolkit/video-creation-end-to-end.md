# AI Avatar Video Creation — SOP
### Women's Nutrition App · Fertility Persona

Have **1 master reference picture of [AVATAR NAME]** ready before starting. All clips must match her exactly.

Use **Google Flow** (0 credits) for all image and video generation.
Use **one Claude/ChatGPT chat for scripting**, a **separate chat for image + video prompting**.

---

## Steps

**1. Script**
In your scripting chat, create the full spoken script for the video. Include hook, key points, and CTA (download the app).

**2. Image Prompting**
In your prompting chat, paste the script and ask for image prompts — one per clip. Minimum 3 images, up to 10+ per video. Always attach the [AVATAR NAME] reference image so she stays consistent.

**3. Video Prompting**
In the same prompting chat, paste all generated images and ask for image-to-video prompts. Each prompt = visual description of what happens in that clip. Then ask for the full spoken script formatted per clip.

**4. Generate in Flow**
Paste the video prompt + spoken text + scene image into Flow. Generate on 0 credits / light priority. If a clip looks wrong, go back to your prompting chat, describe the issue, get an adjusted prompt, and regenerate.

**5. Save Clips**
Number every clip (01, 02, 03…) and save to Drive at 1080p.

**6. Voiceover — ElevenLabs**
Paste each clip's spoken text into ElevenLabs in order. Download all voiceovers and upload to Drive.

**7. Edit — CapCut**
Download all clips and voiceovers from Drive. Edit in CapCut. Add standard captions.
