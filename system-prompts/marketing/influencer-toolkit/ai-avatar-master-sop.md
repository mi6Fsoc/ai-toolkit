## PHASE 2 — PER VIDEO PRODUCTION

Always have the **Master Reference Image** and **Character Sheet** open before starting.

---

**1. Script**
In your scripting chat, write the full spoken script. Structure: Hook → Key points → CTA (download the app).

**2. Image Prompts**
In your prompting chat, paste the script and ask for one image prompt per clip (min 3, up to 10+ per video). Always attach the Master Reference Image. Use the Video Prompt Template below.

**3. Video Prompts**
In the same prompting chat, paste all generated images and ask for image-to-video prompts — one visual description per clip. Then ask for the full spoken script formatted clip by clip.

**4. Generate in Flow**
Paste video prompt + spoken text + scene image into Google Flow. Use 0 credits / light priority. If a clip drifts from the reference or looks wrong — go back to the prompting chat, describe the issue, get an adjusted prompt, regenerate.

**5. Save Clips**
Number every clip (01, 02, 03…). Save to Drive at 1080p.

**6. Voiceover — ElevenLabs**
Paste each clip's spoken text into ElevenLabs in clip order. Download all voiceovers. Upload to Drive.

**7. Edit — CapCut**
Download all clips and voiceovers from Drive. Edit in CapCut. Add standard captions.
