# Create Ad Concept

**Objective**

Create a static ad concept that combines visual composition (layout, art direction, style, elements) with copywriting (headline, tagline, supporting text). The product picture provided by the user must be integrated into the ad creative and remain clearly visible.

**Inputs Provided by User**

- **Brand Context**: Description of the brand, its tone of voice, values, and positioning.
- **Creative Brief**: Target audience, campaign objective, key message, desired emotions, and any mandatory elements (colors, call-to-action, formats, etc.).
- **Product Picture**: This will be placed into the composition later. You do not need to describe the product precisely, only indicate its placement and prominence.
- **Visual references**: Previous ad creatives of the brand. They generated high level of performance and on-brand.

You can use Web search to fill missing information if need be.

**Instructions**

- **Analyze the brand context and visual refrences** to align tone of copywriting and visual direction.
- **Interpret the creative brief** to define the ad’s purpose, target audience, and intended emotion.
- **Generate a static ad concept** including:
- **Visual Composition**: Suggested layout (where the product should appear, positioning of text and graphic elements). Art direction (style, photography/illustration approach, color palette, mood, background elements). Hierarchy of elements (what should catch the eye first, secondary focal points).
- **Copywriting**: Main headline (bold, impactful, aligned with brand voice). Supporting subline (clarifies the value or benefit). Call-to-action (clear and motivating).
- Ensure the **product remains visible** within the creative.

Output should be **concise but precise**, structured as a creative brief for a designer.

Only return this ad concept. Nothing else.
