## General Principles of Superior Prompt Design

Any prompt — whether generated by a human or AI — should adhere to core best practices. These ensure clarity of instruction and reliability of the AI's response:

- **Clarity & Specificity:** State exactly what the AI should do, avoiding ambiguity. A well-defined prompt outlines the task and desired output with precise language[[2]](https://blog.promptlayer.com/ai-prompt-engineering-best-practices/#:~:text=,hero%20discovers%20the%20hidden%20treasure). For example, *"Write a 200-word blog introduction about how AI is used in education, focusing on accessibility for students with disabilities,"* is better than a vague *"Tell me about AI."* Clearly specifying the topic, scope, and format reduces misunderstanding[[2]](https://blog.promptlayer.com/ai-prompt-engineering-best-practices/#:~:text=,hero%20discovers%20the%20hidden%20treasure).
- **Context or Role Specification:** Provide any necessary context or assign the AI a role to guide tone and perspective. For instance, prefixing with *"You are a history professor…"* or *"Act as a financial advisor…"* can help the model adopt the correct voice and domain knowledge[[3]](https://blog.promptlayer.com/ai-prompt-engineering-best-practices/#:~:text=,bullet%20points%20in%20simple%20language). Contextual framing gives the model background to ground its response (e.g. supplying relevant facts or scenarios in the prompt).
- **Structured Format & Guidance:** If a certain output format is needed, instruct it explicitly. Examples include requesting bullet lists, step-by-step instructions, tables, JSON, or sections. You can also structure the prompt itself into parts (e.g. a bullet list of requirements). Combining roles and format guidance yields clarity, as in: *"You are an expert grant writer.* *Task: Draft a 5-point grant proposal outline for project X.* *Format: Provide the output as a numbered list with brief explanations."*[[4]](https://blog.promptlayer.com/ai-prompt-engineering-best-practices/#:~:text=,shot%20deal). This scaffolding helps the model organize its answer.
- **Chain-of-Thought Reasoning:** Encourage the model to work through complex problems stepwise by including phrases like *"Let's think this through step by step"*. This so-called chain-of-thought prompting can improve reasoning on tasks like math or logic puzzles[[5]](https://blog.promptlayer.com/ai-prompt-engineering-best-practices/#:~:text=Advanced%20Techniques%20for%20Maximizing%20AI,Performance). It prompts the AI to **show its reasoning process** before final answers, often leading to more accurate results for complex queries.
- **Examples (Few-Shot Prompting):** Demonstrating the desired pattern with examples can train the model on the fly. For instance, providing a couple of question-answer pairs (for Q&A tasks) or an example input and desired output format will signal how the model should respond[[6]](https://blog.promptlayer.com/ai-prompt-engineering-best-practices/#:~:text=%2A%20Few,are%20essentially%20interactive%20prompt%20chaining). Use this when the task is nuanced or format-sensitive. Even one example can clarify requirements better than description alone.
- **Constraints and Directives:** Clearly state any limitations or special instructions. This can include word limits (e.g. *"in 100 words"*), style guidelines (e.g. *"use a formal tone"* or *"avoid technical jargon"*), or content to avoid (e.g. *"Do not include personal opinions or unverified facts."*[[7]](https://blog.promptlayer.com/ai-prompt-engineering-best-practices/#:~:text=,This)). Constraints help the model understand boundaries and reduce undesired outputs.
- **Output Verification Cues:** Define what a successful output should contain or how it will be evaluated. Many prompts end by describing the expected result (e.g. *"The final output should be a concise summary capturing all key points."*). These act as **evaluation criteria** within the prompt, signaling the model about the qualities of a good answer. For instance, a prompt might say *"ensure the answer is error-free and follows APA format"*, so the model will check those aspects.
- **Ethical and Safety Framing:** When appropriate, include instructions that reinforce ethical guidelines or safe completions. For example, a prompt can remind the model not to produce disallowed content, to be unbiased, or to consider fairness in its response. This *ethical framing* helps align the output with responsible AI use[[8]](https://www.reddit.com/r/PromptEngineering/comments/1hv1ni9/prompt_engineering_of_llm_prompt_engineering/#:~:text=r%2FPromptEngineering%20www,Prompt). An example might be: *"Explain the medical procedure in simple terms and* *do not provide any medical advice beyond general information."* Such qualifiers ensure compliance with safety norms.
- **Iterative Refinement:** Treat prompt design as iterative. If the output isn't optimal, adjust the prompt and try again. For training purposes, you can even include a self-refinement step (meta-prompting) where the model evaluates or improves an initial prompt. The key is to incrementally add clarity, context, or constraints based on the model's last answer. Over time, this yields a refined prompt that consistently produces high-quality results[[9]](https://blog.promptlayer.com/ai-prompt-engineering-best-practices/#:~:text=,in).

These principles form the foundation. The sections below break down prompt patterns by category, capturing the **objectives, structures, and techniques** common to each domain. Each section includes the framework analysis, reusable templates with placeholders, and concrete prompt examples.

## Writing Assistance

### Framework

**Focus & Objectives:** This category covers prompts that assist with writing tasks like emails, essays, outlines, and content ideas. The core objective is typically to produce or plan written content in a coherent, structured, and audience-appropriate manner. Prompts often guide the model through the writing process – from brainstorming to structuring to polishing – for professional or creative pieces.

**Common Structures & Patterns:**

- **Clear Task Definition:** Writing prompts open by specifying the writing piece and purpose, e.g. *"Compose a professional email to address a client's concern about X"* or *"Develop a detailed outline for a persuasive essay on Y."* They clearly articulate the goal (email, essay outline, blog ideas, etc.) and context[[10]](https://promptperfect.xyz/static/cookbook.html#:~:text=Compose%20a%20professional%20email%20to,about%20a%20recent%20project%20deadline)[[11]](https://promptperfect.xyz/static/cookbook.html#:~:text=Your%20task%20is%20to%20develop,to%20create%20a%20coherent%20narrative). This ensures the model knows exactly what kind of writing to produce.
- **Step-by-Step Guidance:** They often break the task into steps: *"Start by... Next, ... Then, ... Conclude by..."* This sequential scaffolding is explicit in many examples[[10]](https://promptperfect.xyz/static/cookbook.html#:~:text=Compose%20a%20professional%20email%20to,about%20a%20recent%20project%20deadline). For instance, a prompt may say: *"Start by acknowledging the reader's concern, then provide an update, and conclude with next steps."* Such ordered instructions help the model systematically cover all parts of the content.
- **Tone and Style Cues:** Writing assistance prompts emphasize tone (professional, engaging, formal, etc.) and voice appropriate to the audience. E.g. *"Maintain a professional and empathetic tone throughout"* for client emails, or *"Use a clear, persuasive tone"* for an essay. The model is directed to adopt a persona implicitly, such as *a professional communicator* in an email or *a knowledgeable author* in an essay. This ensures consistency and audience alignment.
- **Formatting and Structure:** Prompts frequently instruct how to format the output. For example, an essay outline prompt might specify using headings, subheadings, or bullet points for clarity[[12]](https://promptperfect.xyz/static/cookbook.html#:~:text=The%20outline%20should%20be%20clearly,for%20your%20persuasive%20essay%2C%20showcasing). A blog ideas prompt might ask for a numbered list of ideas. By providing a structure (outline, list, sections), the prompt helps the model organize content logically and readably.
- **Length or Content Constraints:** Some writing prompts include constraints like word counts (e.g. a 100-word personal statement) or time length (30-second elevator pitch). Others specify to include or avoid certain content (e.g. *"avoid clichés"*, *"include specific examples"*). These boundaries guide the model's output to meet specific requirements or quality standards (like conciseness or originality).
- **Output Expectations:** Nearly all such prompts conclude by describing the desired outcome or quality of the final piece. For example, *"The final output should be a polished email that reassures the client"*[[10]](https://promptperfect.xyz/static/cookbook.html#:~:text=Compose%20a%20professional%20email%20to,about%20a%20recent%20project%20deadline), or *"The final output should be a comprehensive outline serving as a solid foundation for the essay"*[[13]](https://promptperfect.xyz/static/cookbook.html#:~:text=The%20outline%20should%20be%20clearly,clear%20and%20compelling%20argument%20throughout). This explicit success criterion helps the model understand what a good answer looks like (e.g. *polished*, *well-organized*, *error-free*). It's effectively telling the AI how we'll judge its work, prompting it to self-check before outputting.

### Template

- **Task:** Write a [type of document] about [topic/purpose].
- **Instructions:**
  1. Begin by [first step – e.g., introducing the topic or acknowledging context].
  2. Then, [next step – e.g., provide supporting details, arguments, or updates].
  3. Follow up with [additional step – e.g., evidence, examples, or clarifications].
  4. Conclude by [final step – e.g., summarizing key points or issuing a call-to-action].
- **Tone/Style:** Use a [desired tone] tone throughout, appropriate for [audience or role, e.g., "a formal business email" or "an academic essay"].
- **Format:** Present the content as [format – e.g., paragraphs, bullet points, an outline] for clarity.
- **Constraints:** [Any length limits or specific do's/don'ts, e.g., "150-200 words", "include at least 3 key points", "no jargon"].
- **Expected Output:** A well-written [document type] that [summarize objective, e.g., "addresses the client's concerns and provides reassurance"], demonstrating [key quality, e.g., professionalism, clarity, persuasiveness].

This template can be adapted for many writing scenarios by filling in the document type (email, cover letter, essay, social media post, etc.), the topic or goal, and the specific style/format requirements.

### Prompt Examples

- **Brainstorming blog post ideas:**
  - Prompt: "Generate a list of 10 unique and engaging blog post ideas about [topic], suitable for a [target audience] and optimized for [SEO keywords]."
  - Output: "A list of 10 blog post ideas, including catchy titles, brief descriptions, and relevant keywords."
- **Crafting compelling headlines:**
  - Prompt: "Write 5 alternative headlines for an article about [topic] that are [adjective, e.g., catchy, informative, click-worthy] and appeal to [target audience]."
  - Output: "Five distinct headlines, each with a brief explanation of its appeal and target."
- **Summarizing long articles:**
  - Prompt: "Summarize the key arguments and conclusions of the following article in [number] concise bullet points: [article text]."
  - Output: "A bullet-point summary highlighting the main points and conclusions of the article."
- **Rewriting for clarity and conciseness:**
  - Prompt: "Rewrite the following paragraph to improve its clarity and conciseness, targeting a [reading level, e.g., 8th-grade] audience: [paragraph text]."
  - Output: "A revised paragraph that is clearer, more concise, and easier to understand for the specified audience."
- **Expanding on brief notes:**
  - Prompt: "Expand on the following notes to create a [type of content, e.g., short essay, detailed report section] of approximately [word count] words: [notes]."
  - Output: "A well-structured and coherent piece of writing that elaborates on the provided notes."
- **Generating creative story prompts:**
  - Prompt: "Create 3 unique story prompts that include [genre], [key elements, e.g., a mysterious artifact, a moral dilemma], and [setting]."
  - Output: "Three distinct story prompts, each with a brief description of its core concept and potential plot twists."
- **Developing character backstories:**
  - Prompt: "Develop a detailed backstory for a character named [character name] who is a [character archetype, e.g., reluctant hero, cunning villain] and lives in [setting]."
  - Output: "A comprehensive character backstory, including their origins, motivations, and key life events."

## Research Support

### Framework

**Focus & Objectives:** Research support prompts guide academic or investigative tasks such as literature reviews, formulating research questions, finding sources, summarizing articles, or developing proposals. They aim to produce outputs that are methodical, well-documented, and academically sound. The objective is often to organize information or present findings in a scholarly format.

**Common Structures & Patterns:**

- **Goal Articulation:** Each prompt clearly states the research task, e.g. *"Conduct a thorough literature review on [topic]."* or *"Generate insightful research questions on [topic]."* The scope is defined upfront (specific topic or problem statement)[[14]](https://promptperfect.xyz/static/cookbook.html#:~:text=Conduct%20a%20thorough%20literature%20review,of%20disagreement%20among%20the%20sources)[[15]](https://promptperfect.xyz/static/cookbook.html#:~:text=Develop%20insightful%20research%20questions%20that,to%20allow%20for%20meaningful%20exploration), focusing the model on the research domain and goals (e.g. reviewing current knowledge, identifying gaps, etc.).
- **Methodical Step Guidance:** Research prompts break down the process into systematic steps. For example, a literature review prompt might instruct: *"Define the research question, identify databases and search terms, analyze each source, then synthesize the findings."*[[14]](https://promptperfect.xyz/static/cookbook.html#:~:text=Conduct%20a%20thorough%20literature%20review,of%20disagreement%20among%20the%20sources)[[16]](https://promptperfect.xyz/static/cookbook.html#:~:text=Synthesize%20the%20information%20gathered%2C%20identifying,of%20disagreement%20among%20the%20sources). Another prompt might say: *"List main points and evidence, then summarize implications."* This mirrors real research workflows, guiding the model to emulate scholarly rigor and thoroughness.
- **Structure and Formatting:** Outputs are often expected in structured academic formats — e.g. an organized review with sections, a list of research questions, an annotated bibliography entry per source, etc. Prompts explicitly mention using headings, subheadings, or lists for clarity[[17]](https://promptperfect.xyz/static/cookbook.html#:~:text=Build%20your%20guide%20with%20clear,scenarios%20to%20illustrate%20key%20points). For instance, *"Structure your output as a numbered list of research questions"* or *"Use bullet points for each hypothesis."* By specifying format, the prompt ensures the AI's output is easy to read and aligns with academic conventions (like separating methodology, findings, etc.).
- **Analytical Tone & Objectivity:** The tone is typically formal, objective, and analytical. Prompts remind the AI to maintain an objective voice (e.g. *"refrain from personal opinions"*)[[18]](https://promptperfect.xyz/static/cookbook.html#:~:text=Make%20your%20summary%20as%20a,including%20personal%20opinions%20or%20interpretations) and to critically evaluate content (like assessing credibility of sources or identifying gaps in research). Setting an implicit persona of a *researcher or scholar* helps produce answers that are measured and evidence-focused.
- **Inclusion of Criteria or Constraints:** Some research prompts include specific criteria, such as citation style requirements (*"cite each source in APA format"*), or scope limitations (*"focus on studies from the last 5 years"*). They might also ask for certain content like *"include potential implications or future research directions."* These constraints align the output with what one would expect in a research context (e.g. up-to-date sources, properly cited, specific analysis included).
- **Expected Outcome Description:** As with other categories, research prompts often end by describing the desired output's qualities. For example: *"The final output should be a well-written and insightful literature review that demonstrates understanding of the research landscape."*[[19]](https://promptperfect.xyz/static/cookbook.html#:~:text=controversies%20in%20the%20existing%20research), or *"…a comprehensive guide that empowers individuals to effectively participate in the peer-review process."*[[17]](https://promptperfect.xyz/static/cookbook.html#:~:text=Build%20your%20guide%20with%20clear,scenarios%20to%20illustrate%20key%20points). By outlining these expectations (insightful, comprehensive, well-organized, etc.), the prompt guides the model toward scholarly-quality responses.

### Template

- **Task:** [Research task, e.g., "Conduct a literature review on [specific topic]" or "Develop an annotated bibliography for [topic]"].
- **Instructions:**
  - **Scope Definition:** Start by clearly defining the focus (research question or problem statement) to guide the inquiry.
  - **Gathering Sources:** Identify relevant sources (e.g., scholarly articles, books, credible websites) using [databases/search terms] and list key literature related to the topic.
  - **Analysis:** For each source or idea, summarize the main findings or arguments. Note methodologies, evidence, and any limitations or biases.
  - **Synthesis:** Compare and contrast the information from different sources. Identify common themes, disagreements, and gaps in the current research.
  - **(Optional)** If required, propose next steps (e.g., potential experiments, unanswered questions, or how to apply findings).
- **Format:** Present the information as [desired format – e.g., "a structured review with Introduction, Body, Conclusion", "a bullet-point list of research questions with brief explanations", "annotated bibliography entries with citations and summaries"]. Use clear headings or numbering as needed for organization.
- **Style/Tone:** Use an academic and objective tone. Write in the third person, and avoid personal opinions. Support statements with evidence from sources (and cite accordingly if needed).
- **Constraints:** [Any specific requirements, e.g., "include at least 5 sources", "use APA style citations", "focus on studies from the last decade", "approx. 500 words"].
- **Expected Output:** A thorough and well-organized research output that [key outcome – e.g., "provides a comprehensive overview of the topic", "poses insightful questions for further study", "summarizes current knowledge and identifies research gaps"]. It should demonstrate critical thinking and a deep understanding of the subject.

This template can be tailored to various academic support tasks (reviews, proposals, question generation, etc.) by adjusting the task and format sections.

### Prompt Examples

- **Identifying credible sources:**
  - Prompt: "Identify 3 credible sources for research on [topic], focusing on [type of source, e.g., academic journals, reputable news outlets], and explain why each is reliable."
  - Output: "A list of 3 credible sources with brief justifications for their reliability."
- **Extracting key information:**
  - Prompt: "Extract the main arguments and supporting evidence from the following research paper: [paper text]."
  - Output: "A concise summary of the paper's main arguments and supporting evidence."
- **Synthesizing information from multiple sources:**
  - Prompt: "Synthesize the information from the following two articles on [topic] to provide a comprehensive overview, highlighting areas of agreement and disagreement: [article 1 text], [article 2 text]."
  - Output: "A synthesized overview that integrates information from both articles, noting commonalities and differences."
- **Formulating research questions:**
  - Prompt: "Formulate 5 clear and concise research questions about [topic] that could be explored in a [type of research, e.g., qualitative, quantitative] study."
  - Output: "Five well-defined research questions suitable for the specified research type."
- **Creating annotated bibliographies:**
  - Prompt: "Create an annotated bibliography entry for the following source, including a summary of its content and its relevance to [topic]: [source details]."
  - Output: "A complete annotated bibliography entry for the provided source."
- **Identifying research gaps:**
  - Prompt: "Analyze the provided literature review on [topic] and identify at least 3 potential gaps in current research that warrant further investigation."
  - Output: "A list of 3 identified research gaps with brief explanations of their significance."
- **Developing a hypothesis:**
  - Prompt: "Develop a testable hypothesis for a study investigating the relationship between [variable 1] and [variable 2] in [population]."
  - Output: "A clear, testable hypothesis for the proposed study."

## Learning & Education

### Framework

**Focus & Objectives:** Prompts in this category aim to facilitate learning. They involve creating study aids (summaries, guides, flashcards, quizzes), explaining complex concepts, or developing educational content. The objective is to produce outputs that enhance understanding or help with review and practice, often in an accessible manner for learners.

**Common Structures & Patterns:**

- **Educational Goal Clarity:** Prompts clearly state the learning objective, e.g. *"Create a study guide for an upcoming exam on [subject]"* or *"Explain [complex concept] in simple terms."* The topic or concept to learn is specified, ensuring the model focuses on delivering relevant educational content[[20]](https://promptperfect.xyz/static/cookbook.html#:~:text=Your%20goal%20is%20to%20create,and%20bullet%20points%20for%20clarity)[[21]](https://promptperfect.xyz/static/cookbook.html#:~:text=Your%20goal%20is%20to%20explain,and%20make%20it%20more%20relatable). Often the prompt mentions the target learner or context (exam prep, self-study, teaching a non-expert) to tailor the output level.
- **Breaking Down Information:** These prompts encourage dividing content into digestible parts. For example, *"Identify key topics and provide a summary for each"* or *"Outline each step to solve the problem."* The model is guided to produce structured educational material: step-by-step solutions for math problems[[22]](https://promptperfect.xyz/static/cookbook.html#:~:text=Provide%20a%20step,of%20the%20key%20steps%20involved), bullet-point summaries for chapters[[23]](https://promptperfect.xyz/static/cookbook.html#:~:text=Your%20goal%20is%20to%20summarize,chapter%27s%20main%20takeaway%20or%20implications), or Q&A pairs for quizzes. This scaffolding mirrors how a tutor might teach: breaking complex material into simpler chunks.
- **Use of Examples and Analogies:** There's an emphasis on making content understandable. Prompts often say *"use simple language and avoid jargon"*, and *"include examples or analogies to illustrate the concept."*[[21]](https://promptperfect.xyz/static/cookbook.html#:~:text=Your%20goal%20is%20to%20explain,and%20make%20it%20more%20relatable). For instance, explaining a complex concept might involve a real-world analogy as directed by the prompt. This technique helps the model produce an explanation that is relatable and easy to grasp, aligning with educational best practices.
- **Interactive or Engaging Format:** Many learning prompts specify output formats that engage the learner: flashcards (word on one side, definition on the other)[[24]](https://promptperfect.xyz/static/cookbook.html#:~:text=Create%20flashcards%20that%20aid%20in,further%20enhance%20comprehension%20and%20retention), quizzes (with questions and answers)[[25]](https://promptperfect.xyz/static/cookbook.html#:~:text=Your%20goal%20is%20to%20generate,assessment%20and%20learning), or dialogues for language practice. Instructions like *"Structure as a set of quiz questions with an answer key"* or *"Create flashcards with term on one side and definition on the other"* are common. By defining these formats, the prompt ensures the AI provides an output that learners can directly use as a study tool.
- **Tone and Persona:** The tone is encouraging, clear, and often simplified for accessibility. The model might implicitly take on the persona of a tutor or coach: patient, positive, and explanatory. Prompts might include guidance like *"explain as if to someone with no background in the subject"* or *"use a friendly, encouraging tone to keep the learner engaged."* This ensures the output is pedagogically sound — not just correct, but also approachable.
- **Outcome Emphasis:** Educational prompts often conclude with the intended learning outcome, e.g. *"The final output should be a well-structured study guide that makes exam review efficient"*[[26]](https://promptperfect.xyz/static/cookbook.html#:~:text=The%20final%20output%20should%20be,and%20preparation%20for%20the%20exam) or *"...an explanation that demystifies the concept for a broad audience."*[[27]](https://promptperfect.xyz/static/cookbook.html#:~:text=Structure%20your%20explanation%20in%20a,illustrations%20to%20further%20enhance%20understanding). By stating this, the prompt nudges the AI to double-check that the content meets the educational goal (comprehensive yet concise, accessible, and useful for learning).

### Template

- **Learning Task:** [Description of task, e.g., "Summarize the key concepts of [Chapter/Topic]" or "Create a set of practice quiz questions on [subject]"].
- **Instructions:**
  - Identify the core ideas or skills in [topic] that the learner needs to understand.
  - For each core idea, provide a clear and concise explanation. Use simple language and define any technical terms.
  - Include examples or analogies to illustrate important points (especially for complex concepts).
  - (If creating practice material) Formulate [number] of questions covering these ideas. Vary the format (e.g., multiple-choice, true/false, short answer) as appropriate and provide the correct answers with brief explanations.
- **Format:** Present the material as [desired format – e.g., "a bullet-point study guide", "Q&A flashcards", "numbered quiz questions followed by answers"]. Use headings or numbering to separate sections or topics for clarity.
- **Tone:** Instruct in a patient, encouraging tone, as if you are a friendly tutor. The explanation should be accessible to [learner level, e.g., "a high school student", "a beginner with no background"].
- **Constraints:** [Any limits or specific focuses, e.g., "keep each explanation under 50 words", "include at least one example for each key point", "cover at least 5 major concepts"].
- **Expected Outcome:** A learning resource that helps the user grasp [topic] effectively. It should be well-organized and easy to understand, enabling quick review and reinforcing comprehension of the material.

By adjusting the topic and output format, this template can generate summaries, study guides, practice questions, or explanations across a range of subjects.

### Prompt Examples

- **Explaining complex concepts:**
  - Prompt: "Explain [complex concept] in simple terms, suitable for a [target audience, e.g., 10-year-old, college student] with no prior knowledge of the subject."
  - Output: "A clear and easy-to-understand explanation of the complex concept."
- **Generating quiz questions:**
  - Prompt: "Generate 5 multiple-choice questions about [topic] with 4 answer options each, including the correct answer and a brief explanation for each."
  - Output: "Five multiple-choice questions with answer options, correct answers, and explanations."
- **Creating study guides:**
  - Prompt: "Create a study guide for [subject/chapter] that includes key terms, important concepts, and potential essay questions."
  - Output: "A comprehensive study guide covering key aspects of the subject/chapter."
- **Providing personalized learning paths:**
  - Prompt: "Based on my current understanding of [topic] (beginner/intermediate/advanced) and my learning style (visual/auditory/kinesthetic), suggest a personalized learning path with resources and activities."
  - Output: "A tailored learning path with recommended resources and activities."
- **Simulating historical events:**
  - Prompt: "Describe the events leading up to [historical event] from the perspective of [historical figure], including their motivations and challenges."
  - Output: "A narrative account of the historical event from the specified perspective."
- **Practicing problem-solving skills:**
  - Prompt: "Present a [type of problem, e.g., math, logic] problem related to [topic] and guide me through the steps to solve it, providing hints if I get stuck."
  - Output: "A problem-solving exercise with step-by-step guidance and hints."
- **Facilitating language practice:**
  - Prompt: "Engage in a conversation with me in [language] about [topic], correcting my grammar and vocabulary as we go."
  - Output: "An interactive conversation in the target language with real-time corrections."

## Business & Productivity

### Framework

**Focus & Objectives:** This category addresses professional productivity tasks: writing business emails, planning projects, generating marketing ideas, creating agendas, analyses (SWOT, etc.), and similar workplace or business scenarios. The prompts aim to produce organized and action-oriented outputs that help in professional settings, emphasizing clarity, conciseness, and effectiveness.

**Common Structures & Patterns:**

- **Precise Task Description:** Prompts explicitly state the business task and context, e.g. *"Compose a professional business email to communicate [purpose]"* or *"Create a project timeline for [project name]."* The objective is clear (email, timeline, agenda, etc.) and often includes the key content to cover (like addressing specific concerns in an email, or including tasks and deadlines in a timeline)[[28]](https://promptperfect.xyz/static/cookbook.html#:~:text=Compose%20a%20professional%20business%20email,closing%2C%20followed%20by%20your%20signature)[[29]](https://promptperfect.xyz/static/cookbook.html#:~:text=Create%20a%20detailed%20project%20timeline,or%20checkpoints%20along%20the%20way). This ensures relevance to the business scenario at hand.
- **Sequential Instructions:** Similar to other categories, business prompts break down the content into logical steps or components. For instance, an email prompt might outline: *"Start with a clear subject line, greet politely, state your purpose with necessary details, and end with a call-to-action and professional closing."*[[28]](https://promptperfect.xyz/static/cookbook.html#:~:text=Compose%20a%20professional%20business%20email,closing%2C%20followed%20by%20your%20signature) Another example is a meeting agenda prompt: *"Define meeting objectives, list topics with time allocations, assign presenters, and conclude with next steps."* The ordered checklist format is common, reflecting real business document structures.
- **Professional Tone & Language:** The tone is guided to be formal, clear, and polite/professional. ## Time Management and Organization

**Focus & Objectives:** Time management prompts help users organize their tasks, schedule their time, and increase productivity. Outputs include to-do lists, prioritization frameworks (like Eisenhower matrix), weekly schedules, time-blocking plans, and methods to balance work-life. The output is expected in that structured format (matrix, schedule, list). Prompts explicitly mention the format: *"Structure your output as a matrix/table"* or *"as a weekly calendar"*[[71]](https://promptperfect.xyz/static/cookbook.html#:~:text=Create%20a%20well,for%20flexibility%20and%20unexpected%20events). The tone is usually straightforward and instructional (less emotional, more "let's get it done" attitude, yet encouraging). If the scenario is included (like an overwhelmed person), the prompt might implicitly require a reassuring tone that *"this is manageable if approached systematically."*
- **Goal Reinforcement:** The final expected result is described in terms of clarity and usability: e.g., *"The final output should be a clear and actionable daily to-do list that helps the user stay organized and achieve their objectives."*[[76]](https://promptperfect.xyz/static/cookbook.html#:~:text=Structure%20your%20output%20as%20a,or%20additional%20details%20as%20needed). Or *"...a well-balanced weekly schedule that promotes productivity and personal time."*[[77]](https://promptperfect.xyz/static/cookbook.html#:~:text=Structure%20your%20output%20as%20a,scheduled%20activities%20and%20time%20blocks). ## Problem Solving and Critical Thinking

### Framework

**Focus & Objectives:** These prompts help in tackling complex problems or making decisions. They guide the AI (and user) through analytical methods like root cause analysis, solution brainstorming, pros/cons evaluation, action planning, etc. The objective is to produce a logical breakdown of the problem and potential resolutions, often in a structured, methodical way that aids critical thinking.

**Common Structures & Patterns:**

- **Problem Definition:** Prompts start by clearly restating or focusing the problem. For instance, *"Analyze a complex problem: [describe problem]."* or *"Develop an action plan to solve [specific issue]."* They emphasize understanding the problem fully as a first step[[78]](https://promptperfect.xyz/static/cookbook.html#:~:text=Employ%20root%20cause%20analysis%20to,most%20significantly%20to%20the%20problem)[[79]](https://promptperfect.xyz/static/cookbook.html#:~:text=Develop%20a%20detailed%20action%20plan,challenges%20and%20develop%20contingency%20plans). This ensures the output begins by framing the problem statement, which is crucial for any systematic problem-solving.
- **Analytical Frameworks:** Many prompts explicitly mention a framework or technique: *"Use root cause analysis (5 Whys or fishbone diagram) to identify causes"*[[78]](https://promptperfect.xyz/static/cookbook.html#:~:text=Employ%20root%20cause%20analysis%20to,most%20significantly%20to%20the%20problem)[[80]](https://promptperfect.xyz/static/cookbook.html#:~:text=Employ%20the%20,fundamental%20flaw%20in%20a%20process), *"Use a pros/cons list or decision matrix"*[[81]](https://promptperfect.xyz/static/cookbook.html#:~:text=Conduct%20a%20thorough%20evaluation%20of,each%20factor%20to%20facilitate%20comparison), *"Apply lateral thinking to generate innovative ideas."* The output is then structured accordingly: e.g., a numbered sequence of "Why?" questions and answers for the Five Whys[[80]](https://promptperfect.xyz/static/cookbook.html#:~:text=Employ%20the%20,fundamental%20flaw%20in%20a%20process), or a table listing options with pros and cons[[81]](https://promptperfect.xyz/static/cookbook.html#:~:text=Conduct%20a%20thorough%20evaluation%20of,each%20factor%20to%20facilitate%20comparison). By invoking these tools, prompts lead the AI to follow known logical patterns, resulting in organized and thorough analysis.
- **Multiple Solutions/Brainstorming:** Prompts often ask for more than one solution or idea, not just the "one right answer." E.g., *"Brainstorm creative solutions to overcome obstacle X (provide a list of ideas)."*[[82]](https://promptperfect.xyz/static/cookbook.html#:~:text=Brainstorming%20creative%20solutions%20to%20overcome,obstacles) or *"Develop multiple possible solutions and then evaluate each."* This encourages divergent thinking first, followed by convergent evaluation. The structure might be: list of ideas, then a short evaluation for each, or a recommendation of the best option after listing alternatives.
- **Step-by-Step Reasoning:** There is frequently an element of sequential logic. For example, *"Step 1: define the problem. Step 2: analyze causes. Step 3: generate solutions. Step 4: choose solution. Step 5: plan implementation."* Some prompts directly say *"outline a step-by-step action plan"*[[83]](https://promptperfect.xyz/static/cookbook.html#:~:text=Develop%20a%20detailed%20action%20plan,challenges%20and%20develop%20contingency%20plans) or *"break the big problem into smaller parts."*[[84]](https://promptperfect.xyz/static/cookbook.html#:~:text=Deconstruct%20a%20complex%20or%20overwhelming,on%20their%20importance%20and%20impact). The model's output then mirrors these steps, ensuring no stage of problem-solving is skipped (from understanding to execution). This is essentially chain-of-thought structured explicitly.
- **Objective Tone & Criteria:** The tone is analytical, neutral, and systematic. Prompts dealing with decisions often instruct to remain objective (e.g., use data or clearly list factors). They might request criteria weighting or mention considering various factors (cost, time, impact, risk). For example, *"Assign weights or scores to each option's pros and cons for comparison."*[[81]](https://promptperfect.xyz/static/cookbook.html#:~:text=Conduct%20a%20thorough%20evaluation%20of,each%20factor%20to%20facilitate%20comparison). This drives the AI to not just list factors but also judge them, simulating a rational decision process. The persona here is like a *business analyst or consultant* who is logical and fact-focused.
- **Conclusion/Recommendation:** Many of these prompts end by expecting a recommendation or identification of a root cause, etc., as a result of the analysis. E.g., after analysis, *"The final output should pinpoint the root cause enabling effective solution."*[[78]](https://promptperfect.xyz/static/cookbook.html#:~:text=Employ%20root%20cause%20analysis%20to,most%20significantly%20to%20the%20problem)[[85]](https://promptperfect.xyz/static/cookbook.html#:~:text=Structure%20your%20output%20as%20a,and%20the%20identified%20root%20causes) or *"...an objective evaluation that facilitates informed decision-making."*[[81]](https://promptperfect.xyz/static/cookbook.html#:~:text=Conduct%20a%20thorough%20evaluation%20of,each%20factor%20to%20facilitate%20comparison)[[86]](https://promptperfect.xyz/static/cookbook.html#:~:text=The%20final%20output%20should%20be,making). By stating this, the prompt ensures that the AI doesn't just present analysis but also delivers a clear takeaway or next step (like which solution seems best, or what main issue to address first).

### Template

- **Problem Statement:** [Clearly describe the problem or decision to be made. If complex, include context, e.g., "A product launch is failing due to poor sales. Analyze why and how to improve."].
- **Approach:**
  1. **Problem Definition:** Restate or clarify the problem to ensure understanding. Identify the objectives (what would a solution achieve?).
  2. **Root Cause Analysis:** Break down the problem. [If using 5 Whys or similar:] Ask "Why?" iteratively to dig into potential root causes. List each cause identified. Alternatively, list possible causes in categories (e.g., People, Process, Technology, etc. if using a fishbone diagram approach).
  3. **Idea Generation (Solutions):** Brainstorm several possible solutions or interventions addressing the root causes or ways to achieve the objective. Aim for diverse approaches (including creative or non-obvious ones). List these solutions (Solution A, B, C, ...) with a brief note on how each would resolve the issue.
  4. **Evaluation:** For each solution, outline pros and cons (advantages, disadvantages, potential risks, cost, impact, feasibility). This can be in bullet points or a mini table for each solution. If applicable, mention any criteria being used to judge them (e.g., cost-effectiveness, speed of implementation, long-term viability).
  5. **Recommendation/Action Plan:** Based on the above, recommend the best course of action (or top 1-2 solutions). Explain why this choice addresses the problem effectively. Then, outline a brief action plan to implement the chosen solution: steps to take, who would be responsible (if relevant), and a timeline or immediate next steps.
- **Format:** Ensure the response is well-organized:
  - Possibly use subheadings for each of the above sections (Problem, Causes, Solutions, Evaluation, Recommendation).
  - If listing causes or solutions, use bullet points or numbered lists for clarity.
  - For pros/cons, you might format as an indented list or a table. Keep the structure clear so a reader can scan options and their pros/cons easily.
- **Tone:** Use a logical, neutral tone, like a consultant or analyst reporting findings. It should be fact-based and free of bias or emotion. However, keep it clear and accessible (explain any technical terms if used, since the goal is to make the reasoning easy to follow).
- **Expected Outcome:** A comprehensive breakdown of the problem and potential solutions, leading to a well-justified recommendation. The final output should provide insight into *why* the issue is occurring (root causes) and *how* to address it effectively, giving the user a concrete path forward to resolve the problem.

Adapting this template with the specific problem details will guide the AI to produce a thorough analysis and solution plan for anything from business challenges to personal dilemmas or technical problems.

## Language Learning

### Framework

**Focus & Objectives:** Language learning prompts are designed to help practice a target language. They cover things like learning vocabulary, practicing dialogue, explaining grammar, etc. The objective is to produce educational language content – phrases, translations, conjugation tables, role-play scenarios – that facilitate active practice and understanding for the learner.

**Common Structures & Patterns:**

- **Context and Use-Case Orientation:** Prompts often specify a context for language use. For example, *"Practice common conversational phrases for traveling (ordering food, asking for directions)."* They set a scenario to make the language practice relevant (doctor's appointment, job interview, casual chat, etc.)[[87]](https://promptperfect.xyz/static/cookbook.html#:~:text=Help%20the%20user%20practice%20and,to%20their%20structure%20and%20usage). By doing so, the output phrases or dialogues are situational, which is more useful than random sentences.
- **Lists of Phrases or Examples:** A frequent output format is a list of words or phrases along with their translations or usage. The structure is typically: term -> definition -> example usage. Prompts explicitly ask for such format to be clear, often saying *"Structure as a list of phrases with example dialogues"*[[89]](https://promptperfect.xyz/static/cookbook.html#:~:text=Help%20the%20user%20practice%20and,to%20their%20structure%20and%20usage) or *"provide a table of pronouns and their conjugated forms."*
- **Dialogue Simulation:** Some prompts request a simulated dialogue in the target language with an English translation or with labels. For example, *"Engage in a role-play conversation where one person is a tourist and the other is a local speaking [Language]. Provide both the [Language] dialogue and its English translation."*. This yields an interactive script that learners can study or practice with. The prompts sometimes designate roles and scenario explicitly for clarity[[90]](https://promptperfect.xyz/static/cookbook.html#:~:text=Role)[[91]](https://promptperfect.xyz/static/cookbook.html#:~:text=Structure%20your%20output%20as%20a,key%20phrases%20for%20each%20participant).
- **Grammar Explanation and Tables:** Language learning often involves explaining grammar rules or showing conjugation patterns. A prompt might say *"Explain the rule for using past tense in Spanish and provide a table of conjugation for 5 common verbs."* The output is structured with a brief explanation followed by a table or bullet list of examples. The prompts ensure to mention if they need a particular format (like a table) for clarity[[92]](https://promptperfect.xyz/static/cookbook.html#:~:text=Help%20the%20user%20master%20verb,conjugating%20verbs%20in%20various%20contexts).
- **Encouraging Active Practice:** Some prompts may instruct the AI to formulate exercises or prompt the user. For instance, *"Give the user 5 sentences in English to translate into French (and then provide the correct translations)."* or *"Propose daily language practice routine."*[[93]](https://promptperfect.xyz/static/cookbook.html#:~:text=Establish%20a%20consistent%20daily%20language,partners%2C%20or%20language%20learning%20apps). This shows an interactive or practical component: not just giving information, but prompting the learner to respond or practice, and then giving the answer for self-check.
- **Tone and Clarity:** The tone is instructive but friendly. Simplicity is key: prompts often explicitly say *"use simple language and clear examples."* Explanations should be concise, and steps in reasoning (like a grammar rule) should be broken down clearly. The persona is typically that of a *language tutor*, so the AI's style should be patient and encouraging, making sure not to overwhelm the learner.
- **Outcome Emphasis:** The final output expectation might be phrased as *"a valuable resource for expanding the user's conversational vocabulary"*[[94]](https://promptperfect.xyz/static/cookbook.html#:~:text=Structure%20your%20output%20as%20a,to%20demonstrate%20their%20practical%20application) or *"a practical tool for mastering verb conjugations"*[[95]](https://promptperfect.xyz/static/cookbook.html#:~:text=Structure%20your%20output%20as%20a,the%20chosen%20tense%20or%20mood). In other words, the output should directly aid learning – the prompt implicitly asks: *Is this content directly useful for learning/practice?* If yes, then the prompt's goal is met.

### Template

- **Learning Goal:** [State what the user wants to learn/practice, e.g., "Learn common conversational phrases in [Target Language] for travel", "Practice conjugating verbs in [Target Language] in the past tense", "Expand vocabulary related to [topic] in [Target Language]"].
- **Instructions:**
  - **Context or Scenario:** [If applicable, set a scenario: e.g., "Focus on restaurant and shopping scenarios," "Imagine a conversation at an airport."]. This will make the phrases/dialogues relevant to that situation.
  - **Content Delivery:** Depending on goal:
    - *Phrases/Vocabulary:* Provide a list of useful [language] phrases or words on the topic. For each entry, include:
      - The phrase/word in [Target Language].
      - The English translation (or explanation).
      - (Optional) A short example of the phrase in a sentence or a note on when to use it.
    - *Dialogue:* Present a short dialogue between [Role A] and [Role B] in [Target Language]. Include an English translation or line-by-line interpretation. Clearly label each speaker. (E.g., "Person A ([Language]): [Dialogue]. **Translation:** ...").
    - *Grammar/Conjugation:* Start by briefly explaining the rule (in simple terms). Then provide a mini-table or list of examples. For conjugation: list pronouns and the corresponding verb form. For grammar pattern: give a couple of example sentences demonstrating it.
    - *Exercise:* Optionally, you can include a simple exercise (like fill-in-the-blank or translation challenge) for the user to try, and then provide the correct answers.
  - **Structure:** Organize the output for clarity:
    - Use bullet points or a numbered list for phrases/vocab.
    - Use a dialogue format or table format where appropriate (monospaced font or clear separation for two languages).
    - Use **bold** or italics to highlight the [Target Language] versus the translation, if it helps readability.
  - **Tone:** In the explanatory parts, be encouraging and clear. Assume the reader is a beginner or intermediate, so avoid complicated jargon. The style should feel like a helpful language teacher explaining things step-by-step.
  - **Cultural Note (if relevant):** If certain phrases have cultural usage notes or etiquette (e.g., formal vs informal address), mention it briefly to add context.
- **Expected Outcome:** A set of language learning materials (phrases, dialogues, or explanations) that the user can study to improve their [Target Language] skills. The final output should be educational, easy to follow, and directly useful for practicing the language.

By specifying the target language, topic or scenario, and type of content (phrases, dialogue, grammar), this template can generate a range of helpful language learning prompts, from vocabulary lists to conversational practice scripts.

## Meta-Patterns for Generative Prompt Writing

Designing prompts about prompts – i.e., teaching an AI to create or refine prompts – requires a special meta-awareness. Here we outline patterns for **generative prompt writing**, where the output **is itself a prompt** or instruction set. These techniques help an AI model generate high-quality prompts automatically, essentially turning the model into a *prompt engineer*. Key meta-patterns include:

- **Role Reversal (AI as Prompt Generator):** In the prompt, assign the AI the role of a *prompt engineer or prompt generator*. For example: *"You are an AI prompt creator. Your task is to produce a prompt that... [does X]."* This explicitly tells the model to operate on a meta-level. By giving this role, the AI will frame its output in terms of what a prompt should be. It often results in the AI adopting a more instructional tone, as it knows it's not solving the end-task directly but guiding another to do so.
- **Prompt Goals and Criteria:** Clearly state what the generated prompt should accomplish and any criteria it must meet. This is akin to telling the AI the **specifications for the prompt** it's about to write. For instance: *"Generate a prompt that instructs a user how to write a persuasive essay, including an outline structure, expected tone, and length constraints."* By articulating the goal of the prompt (e.g., instruct the user, include certain details, produce a specific format), the AI has a checklist to satisfy in the prompt it creates[[96]](https://cookbook.openai.com/examples/enhance_your_prompts_with_meta_prompting#:~:text=Meta,4o). Essentially, you give the AI a mini design brief for the prompt.
- **Incorporating Best Practices in the Output:** When the AI writes a prompt, we want it to automatically use the superior design principles outlined earlier (clarity, context, examples, etc.). One way to induce this is by reminding it within the meta-prompt: *"Adhere to prompt engineering best practices: be clear, specify roles, provide format instructions, and include constraints."*[[96]](https://cookbook.openai.com/examples/enhance_your_prompts_with_meta_prompting#:~:text=Meta,4o). This way, the prompt the AI generates will likely contain those elements. For example, the AI's output might be a multi-part prompt with sections for Task, Format, Constraints, etc., because we instructed it to include those features.
- **Self-Refinement Loops:** A powerful meta-pattern is to have the AI refine prompts iteratively. For instance, you can prompt: *"Draft a prompt for [task]. Then critique its weaknesses and produce an improved version."* This makes the AI first output an initial prompt, then analyze it (maybe it realizes it could be clearer or more specific), and finally output a refined prompt. This approach leverages the AI's own chain-of-thought to enhance prompt quality, effectively performing a double-pass. The result is often a more polished prompt that might not have been achieved in a single shot.
- **Few-Shot Meta-Prompting:** Just as examples help in normal prompting, providing examples of good prompts can guide the AI in prompt generation. You can show the AI a couple of prompts (as text) that are considered well-crafted, then ask it to create a new prompt of similar style. This is like giving it a mini prompt cookbook reference. For instance: *"Example Prompt A: [well-structured prompt]. Example Prompt B: [another one]. Now generate a prompt that helps a user summarize an article, following a similar structure and tone as the examples."* The AI will generalize patterns from the examples (like using steps, or final output expectations) and apply them to the new prompt.
- **Meta-Format (Prompt Template Outputs):** When instructing the AI to output a prompt, it's often useful to have it produce **placeholders or template-like structures** instead of hard-coded specifics. For example, the AI's answer can be a prompt with slots: *"Write a [type of document] about [topic] with a tone of [tone]. Begin by..."*. To encourage this, you can say in the meta-prompt: *"Provide the prompt in a generalized template form, using placeholders like [topic] where the user would insert specifics."* The AI then knows to output a prompt pattern rather than a fixed prompt. This is extremely useful for creating **reusable prompt templates** that can be filled in by different users or scenarios.
- **Recursive Prompt Use:** This pattern involves using one prompt to generate another, possibly using different model configurations. For example, using a powerful model to generate or optimize prompts for a smaller model (as described in some advanced usage[[96]](https://cookbook.openai.com/examples/enhance_your_prompts_with_meta_prompting#:~:text=Meta,4o)). While this is more about system design than the prompt content itself, you can simulate it by manually instructing: *"Improve the following prompt for clarity and completeness: [initial prompt]."* The AI will output a revised prompt. This demonstrates how one can have a higher-level orchestrator prompt that results in a better second prompt. Over multiple cycles, the prompt can be iteratively refined.
- **Ethical and Constraint Embedding:** When generating prompts, especially for public or general use, you might want the prompt to include ethical guidelines by default. So your meta-prompt can specify: *"Ensure the prompt you generate instructs the user to avoid any disallowed content and encourages ethical use."* The AI will then include something in the generated prompt about content guidelines or respectful tone, preemptively building in a degree of safety. For example, it might add a line "If the user's request involves sensitive content, proceed with caution and ensure compliance with usage policies," depending on how it interprets the instruction. This pattern ensures **safety considerations propagate** into AI-generated prompts.

By applying these meta-patterns, we create a robust framework for prompt generation. Essentially, we guide the AI to not only solve problems directly, but to output **high-quality prompts** that could themselves be used to get solutions from another AI or a user. This layered approach (AI -> prompt -> AI) leverages the model's knowledge of prompting techniques. Tools like PromptPerfect are examples of this concept in practice, where an AI helps improve prompts for other AI models[[97]](https://promptperfect.xyz/static/cookbook.html#:~:text=Generate%20prompts%20like%20these%20automatically,with%20Prompt%20Perfect).

### Example Meta-Prompt

To illustrate, here's a meta-prompt that combines several of the above techniques, as if we are instructing the model to generate a prompt:

> You are a Prompt Generator AI. Create a clear, effective prompt that guides a user to write a concise summary of an article.
>
> The prompt you generate should:
> - Start by assigning the user a role or context (e.g., "You are a journalist...").
> - Clearly instruct the task (summarizing an article) in detail.
> - Include constraints (like target length, avoiding personal opinions).
> - Outline a step-by-step approach (e.g., read, identify key points, summarize).
> - End by describing what a good summary should achieve (the expected outcome).
> - Maintain an instructive, neutral tone.
>
> Do **not** solve the task (do not write a summary). Only output the prompt itself, formatted as the user should see it, and use placeholders like \<article\> where the article text would be inserted.

The expected output from the model given the above meta-prompt would be a well-structured prompt ready to use for summarizing an article, something like:

> *"You are an experienced journalist tasked with summarizing an article.* *Task:* *Read the following article \<article\> carefully and identify its central thesis and key supporting points. Then write a concise summary in 3-4 sentences, capturing the main ideas in your own words.* *Constraints:* *Do not include any personal opinions or commentary—stick to the article's content. Aim for clarity and brevity, roughly 100 words.* *Output:* *Provide the summary as a single coherent paragraph that accurately reflects the article's core message."*

This output prompt (as generated by the AI following the meta-instructions) demonstrates how the framework's elements come together: it sets context, provides clear instructions, adds constraints, and describes the desired outcome. Such a prompt is high-quality and likely to yield a good result when actually used, which is the ultimate goal of automatic prompt generation.
